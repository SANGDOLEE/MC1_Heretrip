<h1> Heretrip 🛩️ </h1>
<br/>

> **사람들에게 다가가기 어려워하는 여행자** 들이 **가이드를 통한 대화 챌린지**와 **기록**을 통해 **성장** 할 수 있도록 도와주는 앱

<br/>
<br/>

## Feature List 🎮
- 랜덤으로 조건을 제시해 대화 상대 추천기능
- 여러가지 대화 주제를 통해 가이드 제공
- 여행 사진과 텍스트를 기록하고 저장
